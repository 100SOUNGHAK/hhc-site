FIREBASE 연동 설정
처음 한 번만 설정하세요
Firebase 무료 계정을 연결하면 고객이 URL로 계약서를 조회할 수 있습니다
1
Firebase 프로젝트 생성
console.firebase.google.com 접속 → [프로젝트 추가] → 프로젝트 이름 입력 (예: hhc-contracts) → Google 애널리틱스 끄기 → [프로젝트 만들기]
2
Firestore 데이터베이스 생성
왼쪽 메뉴 [Firestore Database] → [데이터베이스 만들기] → 테스트 모드로 시작 → 위치: asia-southeast1 선택 → [완료]
3
웹 앱 등록 및 설정값 복사
프로젝트 개요 → ⚙️ [프로젝트 설정] → [앱 추가] → 웹(</>) → 앱 닉네임 입력 → [앱 등록] → firebaseConfig 값 복사
이 비밀번호로 관리자 페이지에 접근합니다. 분실 시 재설정이 필요합니다.
